## Full Markdown support

Everything you'd expect from GitHub-Flavored Markdown:

1. **Bold**, *italic*, and `inline code`
2. [Links](https://commonmark.org) that open in a new tab
3. Nested lists
   - like this one
   - with sub-items
4. Images, tables, blockquotes, and code blocks

> Write your content the way you already write notes — SliderWeb handles the
> presentation.
