Open in mail button not opening Outlook? Make Outlook your default mail app
  • Windows: Settings → Apps → Default apps → set Outlook for email / mailto.
  • macOS: Apple Mail → Settings → General → Default email reader → Microsoft Outlook (the default is Apple Mail).
  • iOS / iPadOS (Outlook app): install Outlook from the App Store, then Settings → Apps → Default Apps → Email → Outlook.
  • iOS / iPadOS (Apple Mail + Outlook account): Settings → Apps → Mail → Default Account → Outlook — opens Apple Mail, sending from your Outlook account.

Or just use Open in Outlook (web) above — it opens a pre-filled compose tab.
