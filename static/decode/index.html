🛡️ Runs in your browser

Paste any opaque value — encodings are detected and unwrapped layer by layer until something readable comes out:

Base64 / URL-safe Hex URL %-encoding Gzip / zlib JWT Escaped JSON data: URL PEM

Detection is conservative: plain words, paths and IDs that merely look like an encoding are left alone. JWTs can be verified in place — paste the secret or JWK/JWKS into the token card.

Each click wraps the current result in one more layer — stack them the same way the decoder unwraps them (e.g. gzip + Base64).
