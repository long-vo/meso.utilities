## Navigation

| Action | Keys |
| --- | --- |
| Next slide | `→` &nbsp; `Space` &nbsp; `Page Down` |
| Previous slide | `←` &nbsp; `Page Up` |
| First / last | `Home` / `End` |
| Zoom in / out | `+` / `−` &nbsp; (or `Ctrl`/`⌘` + scroll) |
| Overview grid | `O` |
| Fullscreen | `F` |

You can also use the floating control bar at the bottom, or click any empty
space on a slide to move forward. Zoom in to inspect a slide, then **drag to
pan**.
