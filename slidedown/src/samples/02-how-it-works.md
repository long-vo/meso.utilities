## How it works

- Drop a set of **Markdown files** onto the start screen
- **Each file becomes one slide**
- Slides are ordered by **filename**, sorted naturally

So `2-intro.md` comes before `10-summary.md`, just as you'd expect.

Prefix your files with numbers to control the order:

`01-title.md`, `02-agenda.md`, `03-details.md` …
